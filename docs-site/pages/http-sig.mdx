# HTTPSig Signing

How to sign requests using HTTP Message Signatures (RFC 9421) with support for both Ethereum and Arweave wallets.

## Overview

Every request to a pod must be signed using HTTP Message Signatures (RFC 9421). This ensures:

1. **Authentication** — Only the owner can access their pod
2. **Integrity** — Request cannot be tampered with
3. **Non-repudiation** — Signer cannot deny sending the request

Permaweb OS supports multiple signature algorithms based on wallet type:

| Wallet Type | Algorithm | Key Type | Address Format |
|------------|-----------|----------|----------------|
| Ethereum | `eth-personal-sign` | ECDSA (secp256k1) | `0x` + 40 hex chars |
| Arweave | `arweave-rsa-pss-sha256` | RSA-2048 | Base64URL, 43 chars |
| RSA | `rsa-pss-sha256` | RSA-2048 | Configurable |
| ECDSA | `ecdsa-p256-sha256` | ECDSA P-256 | Configurable |

## Wallet Type Detection

The system automatically detects wallet type from the address format:

```typescript
function detectWalletType(address: string): 'ethereum' | 'arweave' | null {
  // Ethereum: 0x followed by 40 hex characters
  if (/^0x[a-fA-F0-9]{40}$/.test(address)) {
    return 'ethereum';
  }
  
  // Arweave: Base64URL, exactly 43 characters (no 0x prefix)
  if (/^[A-Za-z0-9_-]{43}$/.test(address)) {
    return 'arweave';
  }
  
  return null;
}
```

### Examples

| Address | Detected Type |
|---------|---------------|
| `0xABC123def4567890123456789012345678901234` | Ethereum |
| `aNBU7tG1-2-3-4-5-6-7-8-9-0-a-b-c-d-e-f-012` | Arweave |
| `invalid` | null (rejected) |

## Quick Start

### Ethereum Signing

```typescript
import { signatureHeaders } from 'http-message-sig';
import { Wallet } from 'ethers';

async function signRequest(wallet: Wallet, podSubdomain: string, request: any) {
  const headers = await signatureHeaders({
    method: 'POST',
    url: '/verify',
    headers: {
      host: podSubdomain,
      date: new Date().toUTCString()
    }
  }, {
    signer: {
      keyid: wallet.address.toLowerCase(),
      sign: async (data: string) => {
        const signature = await wallet.signMessage(data);
        return Buffer.from(signature.slice(2), 'hex');
      }
    }
  });

  return headers;
}
```

### Arweave Signing

Arweave uses RSA-PSS-SHA256 signatures:

```typescript
import { signatureHeaders } from 'http-message-sig';

async function signRequestArweave(
  walletAddress: string, 
  podSubdomain: string, 
  request: any
) {
  const headers = await signatureHeaders({
    method: 'POST',
    url: '/verify',
    headers: {
      host: podSubdomain,
      date: new Date().toUTCString()
    }
  }, {
    signer: {
      keyid: walletAddress, // Arweave address (43 chars)
      alg: 'arweave-rsa-pss-sha256',
      sign: async (data: string) => {
        // Arweave wallet extension signing
        const signature = await window.arweaveWallet.signMessage(
          new TextEncoder().encode(data),
          { name: 'RSA-PSS', saltLength: 32 }
        );
        return new Uint8Array(signature);
      }
    }
  });

  return headers;
}
```

## Signature Components

### Required Components

| Component | Description | Example |
|-----------|-------------|---------|
| `@method` | HTTP method | `POST` |
| `@path` | Request path | `/verify` |
| `host` | Target host | `abc123.permaweb.live` |
| `date` | Request timestamp | `Wed, 11 Mar 2026 10:30:00 GMT` |

### Signature Header

```
Signature: sig1=:abc123...:
Signature-Input: sig1=("@method" "@path" "host" "date");created=1773155811;keyid="0xabc...";alg="eth-personal-sign"
```

For Arweave:
```
Signature: sig1=:xyz789...:
Signature-Input: sig1=("@method" "@path" "host" "date");created=1773155811;keyid="aNBU7...";alg="arweave-rsa-pss-sha256"
```

## Signing Process

### 1. Create Canonical String

```typescript
function createCanonicalString(request: {
  method: string;
  url: string;
  headers: Record<string, string>;
}): string {
  const lines: string[] = [];
  
  // @method
  lines.push(`"@method": ${request.method}`);
  
  // @path
  lines.push(`"@path": ${new URL(request.url).pathname}`);
  
  // @query (optional)
  const query = new URL(request.url).search;
  if (query) {
    lines.push(`"@query": ${query}`);
  }
  
  // Headers (in order specified in Signature-Input)
  for (const [name, value] of Object.entries(request.headers)) {
    lines.push(`"${name.toLowerCase()}": ${value}`);
  }
  
  return lines.join('\n');
}
```

### 2. Sign the String

#### Ethereum (personal_sign)

```typescript
import { Wallet } from 'ethers';

async function signWithEthereum(
  canonicalString: string,
  wallet: Wallet
): Promise<Buffer> {
  // Ethereum personal_sign
  const signature = await wallet.signMessage(canonicalString);
  
  // Convert hex to bytes (remove 0x prefix)
  return Buffer.from(signature.slice(2), 'hex');
}
```

#### Arweave (RSA-PSS-SHA256)

```typescript
async function signWithArweave(
  canonicalString: string
): Promise<Uint8Array> {
  // Using Arweave wallet extension
  const signature = await window.arweaveWallet.signMessage(
    new TextEncoder().encode(canonicalString),
    { 
      name: 'RSA-PSS', 
      saltLength: 32 
    }
  );
  
  return new Uint8Array(signature);
}
```

### 3. Create Headers

```typescript
interface SignatureHeaders {
  Signature: string;
  'Signature-Input': string;
  Date?: string;
}

async function createSignatureHeaders(
  request: {
    method: string;
    url: string;
    headers: Record<string, string>;
  },
  wallet: Wallet | ArweaveWallet
): Promise<SignatureHeaders> {
  const created = Math.floor(Date.now() / 1000);
  
  // Components to sign
  const components = ['"@method"', '"@path"', '"host"', '"date"'];
  
  // Create canonical string
  const canonical = createCanonicalString(request);
  
  // Sign based on wallet type
  const walletType = detectWalletType(wallet.address);
  
  let signature: Buffer | Uint8Array;
  let alg: string;
  
  if (walletType === 'ethereum') {
    signature = await signWithEthereum(canonical, wallet);
    alg = 'eth-personal-sign';
  } else if (walletType === 'arweave') {
    signature = await signWithArweave(canonical);
    alg = 'arweave-rsa-pss-sha256';
  } else {
    throw new Error('Unsupported wallet type');
  }
  
  const signatureB64 = signature.toString('base64');
  
  // Build headers
  const signatureInput = `sig1=(${components.join(' ')});created=${created};keyid="${wallet.address}";alg="${alg}"`;
  const signatureHeader = `sig1=:${signatureB64}:`;
  
  return {
    'Signature': signatureHeader,
    'Signature-Input': signatureInput,
    'Date': request.headers.date
  };
}
```

## Verification (Server-Side)

The sidecar verifies signatures differently based on wallet type:

### Ethereum Verification

```typescript
import { verifyMessage } from 'ethers';

function verifyEthereumSignature(
  canonicalString: string,
  signature: Uint8Array,
  keyId: string
): boolean {
  try {
    const signatureHex = `0x${Buffer.from(signature).toString('hex')}`;
    const recoveredAddress = verifyMessage(canonicalString, signatureHex);
    return recoveredAddress.toLowerCase() === keyId.toLowerCase();
  } catch {
    return false;
  }
}
```

### Arweave Verification

```typescript
import { createPublicKey, verify, constants } from 'node:crypto';

async function verifyArweaveSignature(
  canonicalString: string,
  signature: Uint8Array,
  keyId: string
): Promise<boolean> {
  // Resolve public key from Arweave gateway
  const jwk = await resolveArweavePublicKey(keyId);
  
  if (!jwk) {
    return false;
  }
  
  // Convert JWK to PEM
  const publicKeyPem = jwkToPem(jwk);
  
  // Verify RSA-PSS-SHA256 signature
  return verify('sha256', Buffer.from(canonicalString), {
    key: publicKeyPem,
    padding: constants.RSA_PKCS1_PSS_PADDING,
    saltLength: constants.RSA_PSS_SALTLEN_DIGEST
  }, Buffer.from(signature));
}
```

### Public Key Resolution

For Arweave wallets, the public key is resolved from the Arweave gateway:

```typescript
async function resolveArweavePublicKey(address: string): Promise<ArweaveJWK | null> {
  // Query Arweave gateway for wallet's last transaction
  const response = await fetch(`https://arweave.net/tx/${address}`);
  
  if (!response.ok) {
    return null;
  }
  
  const tx = await response.json();
  
  // The owner field is the base64url-encoded modulus (n)
  // The exponent (e) is typically AQAB (65537)
  return {
    kty: 'RSA',
    n: tx.owner,
    e: 'AQAB'
  };
}
```

## Supported Algorithms

| Algorithm | Description | Key Type | Use Case |
|-----------|-------------|----------|----------|
| `eth-personal-sign` | Ethereum personal sign | Ethereum address | MetaMask, ethers.js |
| `arweave-rsa-pss-sha256` | RSA-PSS with SHA-256 | Arweave JWK | Arweave wallets |
| `rsa-v1_5-sha256` | RSA PKCS#1 v1.5 with SHA-256 | RSA key | Legacy systems |
| `rsa-pss-sha512` | RSA PSS with SHA-512 | RSA key | High security |
| `ecdsa-p256-sha256` | ECDSA with P-256 curve | ECDSA key | Standard ECDSA |
| `ecdsa-p384-sha384` | ECDSA with P-384 curve | ECDSA key | High security |

## Replay Protection

### Nonce

Each request must have a unique `id` field:

```typescript
import { randomUUID } from 'crypto';

const request = {
  id: `req_${Date.now()}_${randomUUID()}`,
  method: 'message',
  params: { content: 'Hello' }
};
```

### Timestamp

Requests must include a fresh timestamp:

```typescript
const headers = {
  ...request.headers,
  date: new Date().toUTCString()
};
```

### Server Validation

```typescript
const MAX_SKEW = 5 * 60 * 1000; // 5 minutes in milliseconds

async function validateReplay(request: Request, nonceStore: Set<string>): Promise<boolean> {
  const body = await request.clone().json();
  
  // Check nonce hasn't been used
  if (nonceStore.has(body.id)) {
    return false;
  }
  
  // Check timestamp is within allowed window
  const requestDate = new Date(request.headers.get('date') || '');
  const now = new Date();
  const skew = Math.abs(now.getTime() - requestDate.getTime());
  
  if (skew > MAX_SKEW) {
    return false;
  }
  
  // Mark nonce as used
  nonceStore.add(body.id);
  
  return true;
}
```

## Complete Example

### Ethereum

```typescript
import { signatureHeaders } from 'http-message-sig';
import { Wallet } from 'ethers';

async function sendSignedMessage(
  wallet: Wallet,
  podSubdomain: string,
  content: string
): Promise<Response> {
  const request = {
    id: `req_${Date.now()}_${crypto.randomUUID()}`,
    method: 'message',
    params: { content },
    timestamp: Date.now()
  };

  // Create signature headers
  const sigHeaders = await signatureHeaders({
    method: 'POST',
    url: '/verify',
    headers: {
      host: podSubdomain,
      date: new Date().toUTCString(),
      'content-type': 'application/json'
    }
  }, {
    signer: {
      keyid: wallet.address.toLowerCase(),
      sign: async (data: string) => {
        const sig = await wallet.signMessage(data);
        return Buffer.from(sig.slice(2), 'hex');
      }
    }
  });

  // Send request
  const response = await fetch(`https://${podSubdomain}/verify`, {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Signature': sigHeaders.Signature,
      'Signature-Input': sigHeaders['Signature-Input'],
      'Date': sigHeaders.Date!
    },
    body: JSON.stringify(request)
  });

  return response;
}
```
